# Data Availability and Reproducibility Statement

This anonymous supplementary package contains the credential-free materials needed to audit the paper's main empirical claims: generated items, the complete API main judging matrix, the current 22-questionnaire human-audit layer, auxiliary web-interface judge scores, controlled A/B diagnostics, redacted API audit summaries, prompt templates, rubrics, model metadata, and robustness checks.

The package intentionally excludes credential material, full raw API logs, provider-sensitive traces, raw survey workbooks, participant-identifying fields, withheld item-mapping files, and author-side workflow notes. Human-audit raters and reader-study participants are represented only through anonymized IDs or aggregate tables.

For web-interface generation and judging, hidden system prompts, session state, decoding settings, and per-call UI metadata were not available from the providers and are reported as not recorded rather than reconstructed. The official API layer is the revised manuscript's main automated evidence; web-interface outputs are retained only as an auxiliary ecological comparison.
