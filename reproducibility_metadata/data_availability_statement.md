# Data Availability and Reproducibility Statement

This anonymous supplementary package contains the credential-free materials needed to audit the paper's main empirical claims: generated items, web-judge scores, professional-validation summaries, controlled A/B diagnostics, official API replication summaries, prompt templates, rubrics, model metadata, and robustness checks.

The package intentionally excludes credential material, raw API logs, provider-sensitive traces, raw survey workbooks, participant-identifying fields, withheld item-mapping files, and author-side workflow notes. Domain-qualified validators and reader-study participants are represented only through anonymized IDs or aggregate tables.

For web-interface generation and judging, hidden system prompts, session state, decoding settings, and per-call UI metadata were not available from the providers and are reported as not recorded rather than reconstructed. The official API replication is included as an auditable robustness check, not as an exact rerun of the original web-interface study.
