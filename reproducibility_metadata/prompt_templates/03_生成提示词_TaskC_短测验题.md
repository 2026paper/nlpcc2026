# Task C 生成提示词：短公众理解测验题

你是一位科学教育测验题设计者。请为下面 30 个“容易被误解的科学概念”分别设计 1 道极短单选题。

## 严格要求

1. 只输出 JSON 数组，不要 Markdown，不要前言，不要总结。
2. 每个概念只输出 1 个 JSON 对象。
3. `question` 不超过 55 个中文字符。
4. 每个选项不超过 18 个中文字符。
5. 必须且只能有 1 个正确答案。
6. 至少 2 个错误选项要对应常见误解。
7. `explain` 不超过 45 个中文字符。

## 输出格式

```json
[
  {
    "id": "C01",
    "concept": "惯性",
    "question": "公交急刹时，人为什么会前倾？",
    "options": {
      "A": "身体保持原运动",
      "B": "前方有吸力",
      "C": "司机推了乘客",
      "D": "重力突然变大"
    },
    "answer": "A",
    "explain": "惯性使身体短时保持原来的运动状态。"
  }
]
```
