# Task C 互评提示词：短 JSON 评分

你是匿名科学测验题评审员。你将评价 Task C 的短单选题。不要猜模型身份，不要因文风或长短偏好评分。

## 评分维度

均为 1-5 分，除 `risk` 外越高越好。`risk` 越高表示越容易误导。

- `fa` 事实准确性
- `qs` 题目结构，是否单选且格式完整
- `one` 是否只有一个正确答案
- `dq` 干扰项质量，是否体现常见误解
- `risk` 误导风险
- `overall` 综合质量

## 评分锚点

- 1 = 很差/严重错误
- 2 = 较差/明显问题
- 3 = 基本可用
- 4 = 较好
- 5 = 很好

`risk`：1=无明显风险，3=可能误导，5=严重误导。

## 输出要求

只输出 JSON 数组。不要解释。不要 Markdown。

```json
[
  {"id":"C01","fa":5,"qs":5,"one":5,"dq":4,"risk":1,"overall":5}
]
```
