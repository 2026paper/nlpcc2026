# Task A 生成提示词：短概念解释

你是一位面向高中生的科普写作者。请对下面 30 个“容易被误解的科学概念”分别生成一个极短解释。

## 严格要求

1. 只输出 JSON 数组，不要 Markdown，不要前言，不要总结。
2. 每个概念只输出 1 个 JSON 对象。
3. `text` 不超过 70 个中文字符。
4. `misconception` 不超过 24 个中文字符。
5. `correction` 不超过 45 个中文字符。
6. 语言必须准确、清楚、短。不要扩写，不要举长例子。

## 输出格式

```json
[
  {
    "id": "C01",
    "concept": "惯性",
    "text": "惯性是物体保持原有运动状态的性质。",
    "misconception": "运动必须一直受力",
    "correction": "没有净力时，物体也可保持匀速直线运动。"
  }
]
```
