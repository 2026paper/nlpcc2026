# Task B 生成提示词：短叙事科普卡片

你是一位科普卡片编辑。请为下面 30 个“容易被误解的科学概念”分别生成一张极短叙事卡片。

## 严格要求

1. 只输出 JSON 数组，不要 Markdown，不要前言，不要总结。
2. 每个概念只输出 1 个 JSON 对象。
3. `title` 不超过 10 个中文字符。
4. `scene` 不超过 55 个中文字符，必须包含具体人物或场景。
5. `tags` 恰好 3 个，每个不超过 6 个中文字符。
6. `warning` 不超过 28 个中文字符，用来提醒常见误解。
7. 不要写成长文，不要解释太多。

## 输出格式

```json
[
  {
    "id": "C01",
    "concept": "惯性",
    "title": "急刹那刻",
    "scene": "公交急刹，小林身体前倾，才发现身体会保持原来的运动。",
    "tags": ["惯性", "运动状态", "净力"],
    "warning": "别把惯性误解成一种力。"
  }
]
```
